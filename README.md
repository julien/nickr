Requirements
------------

  + Download [Go](http://golang.org/dl)

  + Setup your Go environment

  + Install the Go dependencies

    ```shell
    go get github.com/tools/godep
    go get github.com/melvinmt/firebase
    ```

Running the api
---------------

  + In a terminal, issue the following command

    ```shell
    go run main.go
    ```

  + To run tests, do

    ```shell
    go test --coverage ./...
    ```
